NDVI Demo

The demo requests Sentinel-derived NDVI for the visible viewport through a Cloudflare Worker, then renders the result as an analytical surface. Surface height represents NDVI — not elevation.

Navigate to an area
and click Analyse

This demo first requests live Sentinel-derived NDVI and RGB context from a Cloudflare Worker. The browser sends only viewport bounds and an analysis date; Sentinel credentials stay outside the client. If the viewport is too large, the Worker is unavailable, or the imagery request fails, the surface falls back to a synthetic fixture seeded by the same viewport and analysis date.
