Terrain renderer

The same live NDVI data, rendered as terrain. The analytical surface rises above a satellite imagery base plane assembled from map tiles for the current bounds — vegetation index height and aerial context in one view. Orbit controls let you rotate and inspect from any angle; the base map toggle removes the satellite texture.

Scene Pending

Run analysis to view

NDVI -1 0 +1

This renderer uses the same live data path as the Plotly demo — same Cloudflare Worker, same Sentinel-2 source — but renders through Three.js rather than Plotly. The map selects the area; the rendered view replaces it with a terrain mesh above a satellite tile composite for the same bounds.
