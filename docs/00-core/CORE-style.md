# Core Content and Interaction Style Rules

1. Preserve canonical model terms where ontology matters.
2. Prefer neutral system-state language in live readouts.
3. Keep labels stable once introduced.
4. Ensure controls are keyboard-focus visible and stateful with ARIA where relevant.
