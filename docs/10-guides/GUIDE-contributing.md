# Contributing Guide

## Flow

1. Clarify scope.
2. Implement minimal change.
3. Add/update tests.
4. Update affected docs.
5. Run required checks.

## PR minimum

- clear change summary
- affected files list
- test evidence
- docs updated if behavior/semantics changed
