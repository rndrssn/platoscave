GANTT meets Game of Life

This simulation is an abstract comparison between two management logics. The Gantt scaffold represents a Tayloristic, plan-first model with linear phases and fixed dependencies. The Game of Life layer represents local interaction and feedback. As feedback accumulates, parts of the plan can hold while other parts are rewritten.

What This Is Saying

In hard predictive project models, weak signals are often treated as noise until late, and plans are managed as if the system were mostly closed. That can work for complicated work, where cause and effect can be analysed and stabilised. Complex product work is different: open, coupled, and continuously perturbed by users, markets, operations, and adjacent teams.

Planning still matters. The failure mode is treating first assumptions as fixed. Once work meets reality, feedback loops can amplify and reshape outcomes faster than static phase logic can adapt. For any chosen seed, this simulation is deterministic: the same initial state yields the same trajectory. This is a playful abstraction of that shift.
