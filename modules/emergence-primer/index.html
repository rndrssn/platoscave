Conway's Game of Life

Conway's Game of Life is a discrete mathematical system on a two-dimensional grid. Each cell is either alive or dead, and time advances in synchronized steps. At each step, a cell changes state based only on its eight immediate neighbours: birth on exactly 3 neighbours, survival on 2 or 3. This shorthand is written as B3/S23.

Math and Science View

Formally, each update is deterministic: the next state of a cell depends only on its current state and the count of its eight neighbours. Birth happens on exactly 3 neighbours. Survival happens on 2 or 3. All other cases go to dead. There is no randomness in the rule itself.

The scientific interest is that simple local rules create rich global behaviour: stable structures, oscillators, moving patterns, long transients, and sensitivity to initial conditions. So this is a compact model of emergence: complexity from local interaction, not from central control.
